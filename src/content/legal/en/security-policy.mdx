# Security Policy (RFC 9116)

Security and system integrity are paramount for our strategic analysis environment. 
This page outlines the procedures for reporting vulnerabilities related to this domain.

## Reporting
If you have identified a security vulnerability, please report it by sending an email to:
**tiziano@linguecontiziano.com**

## Guidelines
* Provide sufficient technical details to reproduce the issue.
* Do not exploit the vulnerability for malicious purposes or to access unauthorized data.
* Allow us a reasonable amount of time to analyze and resolve the issue before any public disclosure.

## Security File
The machine-readable version of this policy is available at:
`/.well-known/security.txt`
