# Política de Seguridad (RFC 9116)

La seguridad de los sistemas y la integridad de los análisis estratégicos son prioridades absolutas. 
Esta página describe los procedimientos para informar sobre vulnerabilidades relacionadas con este dominio.

## Informes
Si ha identificado una vulnerabilidad de seguridad, infórmenos enviando un correo electrónico a:
**tiziano@linguecontiziano.com**

## Directrices
* Proporcione detalles técnicos suficientes para reproducir el problema.
* No explote la vulnerabilidad con fines maliciosos o para acceder a datos no autorizados.
* Danos un tiempo razonable para analizar y resolver el problema antes de cualquier divulgación pública.

## Archivo de Seguridad
La versión legible por máquina está disponible en:
`/.well-known/security.txt`
