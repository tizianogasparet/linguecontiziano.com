# Politica di Sicurezza (RFC 9116)

La sicurezza dei sistemi e l'integrità delle analisi strategiche sono priorità assolute. 
Questa pagina descrive le procedure per segnalare vulnerabilità riguardanti questo dominio.

## Segnalazioni
Se hai individuato una vulnerabilità di sicurezza, ti prego di segnalarla inviando un'email a:
**tiziano@linguecontiziano.com**

## Linee Guida
* Fornisci dettagli tecnici sufficienti per riprodurre il problema.
* Non sfruttare la vulnerabilità per scopi malevoli o per accedere a dati non autorizzati.
* Concedimi un tempo ragionevole per analizzare e risolvere il problema prima di qualsiasi divulgazione pubblica.

## File di Sicurezza
Il file machine-readable è disponibile all'indirizzo:
`/.well-known/security.txt`
